# Momentum

Know when your music is moving.

Momentum is a mobile-first growth tracking app for independent music artists.

The app helps artists track:

* audience growth
* engagement
* release traction
* cross-platform momentum

across platforms like:

* Spotify
* YouTube
* Instagram
* SoundCloud

## Product Philosophy

Momentum should feel:

* premium
* emotionally rewarding
* visually clean
* motivating
* simple to understand

This is NOT an enterprise analytics platform.

The experience should feel closer to:

* Spotify Wrapped
* Apple Fitness
* Duolingo
* Cash App

than traditional analytics dashboards.

# Development Priorities

## Phase 1

Frontend shell only:

* onboarding
* dashboard
* navigation
* profile/settings
* reusable UI components

Use mock data initially.

## Phase 2

Firebase integration:

* auth
* user accounts
* notifications
* persistence

## Phase 3

API integrations:

1. Spotify
2. YouTube
3. Instagram

## Phase 4

Rules engine:

* momentum score
* insight generation
* traction detection

# Important Notes

* Keep the app lightweight and mobile-first.
* Prioritize UX clarity over analytics complexity.
* Avoid overwhelming dashboards.
* Focus on emotional motivation and simplicity.
